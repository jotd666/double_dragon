# double_dragon
port of Double Dragon for Amiga

Requires a 3-button joystick, CD32 pad, or the keyboard

Features:

- 25 FPS for OCS,ECS and AGA amigas

Note:

AGA version runs on A1200 with sme fastmem: full sfx & music, arcade approaching colors

Credits:

- jotd: reverse-engineering, 68000 transcode, graphics conversion
  sound conversion for the Amiga.
- no9: music
- PascalDe73: icon

Instructions:

5: insert coin
1/2: start game
arrows/joystick: move
red/ctrl: kick
blue/alt: jump
green/space: punch
